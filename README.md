# 🚀 FocusFlow

FocusFlow is a full-stack Django web application designed to help users track, analyze, and improve their study habits through an interactive dashboard and real-time data insights.

---

## 📌 Features

- 🔐 User Authentication (Signup / Login / Logout)
- 📊 Study Tracking Dashboard
- 📅 Last 7 Days Study Analytics
- 📈 Subject-wise Time Distribution
- 🔄 Add & Reset Study Sessions (API-based)
- ⚡ Dynamic Data Visualization

---

## 🧠 Tech Stack

| Layer       | Technology        |
|------------|------------------|
| Frontend   | HTML, CSS        |
| Backend    | Django (Python)  |
| Database   | SQLite           |
| Deployment | Render           |
